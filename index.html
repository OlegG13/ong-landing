Problem: The current 1FA (single factor- password) and 2FA (two factor- password, FOB or Biometrics) often timefail, are hacked, or otherwise fail (link to examples). There are dozens of different varieties of two-factor now, in fact, it is difficult to catalog them all. Some send verification codes over SMS text, while others use email or more hardened verification apps like Duo and Google Authority.

Solution: 4FA (four-factor authentication) uses encrypted keys, facial recognition, voice biometrics, Memorized Segment, and a signature block stored on the WizeBit Global Network to offer seamless, top-tier data protection. The WizeBit model uses possession, inherence and location in its multi-factor authentication process. 4FA systems are still rare, and typically confined to high-security corporate and government installations. For the first time, WizeBit brings these advances to the wider public.

WizeShadow

WizeShadow works by having a user identify their favorite portion of their favorite song. This authentication method is easy to use and is immune to brute force attacks. The user will first have to identify their favorite song and favorite eight second portion of that song. This will serve as the digital passkey. When needed the song will have to be chosen (API connect to a user defined source - (Spotify, soundcloud etc) and then the user will place an 8 second slider bar over the matching part of the song. This will serve as authentication and allow a user to unlock their account.

WizeShadow allows for authentication and account recovery in impaired environments or when biometric features have irrevocably changed (Facial burns, plastic surgery, etc). This authentication innovation allows Wizebit to safeguard authentication while enabling the recovery of forgotten passwords and damaged biometric profiles. WizeBit brings control and ease to the consumer and the enterprise.

Technology
Personal file storage controlled by you. When utilizing WizeBits 4FA, users will be provided access to the WizeBit LockBox. This is a cloud based file storage system running on top of the WizeBit Network.
This allows for secure sharing of personal data files with the assurances of file resiliency (native blockchain file system ensures resilience), reliability (files sharded over a massively sharded global infrastructure) and security (files systems protected by 4FA.)
File access and Control
Traditionally cloud based storage offerings have challenges with authentication and security. WizeBits combination of 4FA and Blockchain technology mitigates these issues. With the WizeBit data provenance model ownership of data is never shared. Your data is always sacrosanct and even WizeBit cannot access it. Due to the nature of how the data is stored, only the user can authorize data access and always retains ownership of their data. Passwords can be directly exposed as the authentication system always relies on multiple additional factors for authentication.

ENCRYPTION OF FILE SYSTEM

Native encryption of sharded data segments ensures data security with data at rest and while in motion. Data is first encrypted and then split into chunks. These chunks are then distributed onto a massively distributed, global network where they are natively verified using the WizeBit Blockchain.
Once the files are summoned the sharded chunks are reassembled and decrypted with the WizeBit client. Any file based attack would have to directly identify the shard chunks attached to a file, be allowed access to all sharded chunks, reassemble the file chunks and break the native file encryption to comprise the system. All this would have to occur, while their attack is being directly memorialized in the WizeBit blockchain.

TIME CONTROLLED ACCESS

In addition to the having an attacker identifying all the shard chunks making up a file, getting the shard chunks, decrypting the file chunks and then reassembling the file chunks we have also introduced time controls on the access to encrypted file chunks. For authorized users this means file access can be limited to a time range or access completely revoked after a particular time.
